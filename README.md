# projetoContato
<br>
Foi desenvolvido em C++, utilizando getters, setters e construtores um programa que permita a digitação (e posterior exibição), dos dados relativos à classe "Contato" abaixo exibida:
<br><br>
<div align="center">
  <pre>
    ---------------------
    | Data              |
    |-------------------|
    | - dia: int        |
    | - mes: int        |
    | - ano: int        |
    ---------------------
  </pre>
  <br>
  <pre>
    -----------------------
    | Contato             |
    |---------------------|
    | - email: string     |
    | - nome: string      |
    | - telefone: string  |
    | - dtnasc: Data      |
    |---------------------|
    | + idade(): int      |
    -----------------------
  </pre>
</div>
